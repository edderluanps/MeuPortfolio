Sobre mim

Eu sou Edder Luan e sou

Minha jornada na área de desenvolvimento de sistemas começou em 2019, quando comecei a cursar Sistemas de informação na Unifacol, onde tive os primeiros contatos com linguagens de programação como Python e Java. Em 2020 resolvi me aprofundar no Java como minha principal linguagem de programação. A partir dai, tive contato com diferentes modos de desenvolvimento, Desktop, Web, Mobile, banco de dados SQL e NoSQL, além de alguns frameworks fortes no mercado, como Hibernate, JSF e Spring. Em 2021 vendi meu primeiro projeto, um software PDV Desktop. Desde então, resolvi me dedicar no desenvolvimento de aplicações Full Stack. Ainda em 2021 tive contato com Angular , React e Ionic, visando integrar alguns projetos Back-end desenvolvidos com java a uma interface de usuário também desenvolvida por mim. A partir daí comecei a direcionar meus estudos nestas Stacks.
